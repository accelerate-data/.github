# Accelerate Data

We build **VibeData** -- an AI-native data platform that turns natural-language
intent into governed, production-ready data pipelines.

## Key Repositories

| Repository | Description |
|-----------|-------------|
| [vd-studio](https://github.com/accelerate-data/vd-studio) | Core product -- visual pipeline builder and AI agent runtime |
| [plugin-marketplace](https://github.com/accelerate-data/plugin-marketplace) | Claude Code plugins for engineering, docs, and review workflows |
| [migration-utility](https://github.com/accelerate-data/migration-utility) | Data migration tools for platform onboarding |

## How We Work

- **AI-assisted development** -- Claude Code agents participate in every stage
  from design to code review.
- **Governance as code** -- standards, rules, and review policies live in the
  repo alongside the code they govern.
- **Linear-driven workflows** -- every change traces back to a Linear ticket,
  from creation through implementation to close.
