## How We Work

- **AI-assisted development** -- Claude Code agents participate in every stage
  from design to code review.
- **Governance as code** -- standards, rules, and review policies live in the
  repo alongside the code they govern.
- **Linear-driven workflows** -- every change traces back to a Linear ticket,
  from creation through implementation to close.
